# psmqte 0.1.1

- Improved `plot.psmqte()` with a monochrome, print-friendly economics-journal style.
- Added confidence-interval plotting when standard errors are available.
- Added documentation and tests for `plot.psmqte()`.

# psmqte 0.1.0

- Initial package skeleton.
- Added `psmqte()` for propensity-score matching QTE/QTT estimation.
- Added `psmate()` for companion ATE/ATT estimation.
- Added matching, balance, simulation, plotting, and summary helpers.
- Added a self-contained synthetic demo dataset and `psmqte_example_data()`.
